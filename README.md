# Ohjelmistotuotanto-kurssin harjoitustyö

## Suorittaminen

Sovellus voidaan suorittaa käyttäen Mavenia seuraavalla komennolla,
joka kääntää sovelluksen ja käynnistää sen graafisen käyttöliittymän:

```shell
mvn compile exec:java
```
